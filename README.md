# Crypto Analysis Panel (React + Vite)

This project is a web-based cryptocurrency analysis application that allows users to
track real-time crypto prices and receive BUY / SELL decision support based on
technical indicators.

The application is built using React and Vite to provide a fast, modern and scalable
frontend architecture.

## Features

- View real-time cryptocurrency prices in USDT and TRY
- Perform technical analysis on selected cryptocurrencies
- Generate BUY / SELL signals
- View analysis history
- Clean and modern user interface
- Login and register system
- Infrastructure suitable for plan-based access

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3
- Optional Node.js and Express backend
- Optional JSON based persistent storage

## React + Vite

This project uses Vite to provide a minimal setup for getting React working with Hot
Module Replacement and fast refresh during development.

Currently, two official Vite plugins are available:

- @vitejs/plugin-react  
  Uses Babel or oxc for Fast Refresh

- @vitejs/plugin-react-swc  
  Uses SWC for Fast Refresh

## Installation

1. Clone the repository
   git clone <repository_url>
   cd <project_folder>

2. Install dependencies
   npm install

3. Start the development server
   npm run dev

4. Open the application in your browser
   http://localhost:5173
